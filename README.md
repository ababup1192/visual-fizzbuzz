# Preparation

```
$ brew install yarn

$ yarn
$ yarn run typings install
```

# Testing
```
$ yarn test
```

# Launch Develop Server

```
$ yarn start
$ open http://localhost:8080/webpack-dev-server/
```

# Build
```
$ yarn build
```
